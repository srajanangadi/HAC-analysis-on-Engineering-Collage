# Ship Classification using Naive Bayes

## Project Overview

This project applies the **Naive Bayes classification algorithm** to analyze ship-related data and predict the target class based on available features. The goal is to build a simple, interpretable classification model for business analytics use cases.

## Business Objective

The objective of this project is to:

* Classify ship records based on given attributes
* Understand patterns in ship-related data
* Build a probability-based predictive model
* Support data-driven decision-making using classification analytics

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Naive Bayes
* Jupyter Notebook

## Dataset

The dataset contains ship-related records with multiple input features and a target variable used for classification. The data was analyzed and prepared for machine learning modeling.

## Project Workflow

1. **Data Loading & Understanding**
   Loaded the dataset and reviewed its structure, feature types, and target variable.

2. **Data Preprocessing**
   Checked missing values, handled categorical variables, encoded features, and split the data into training and testing sets.

3. **Exploratory Data Analysis**
   Analyzed feature distributions, target class balance, and important patterns in the dataset.

4. **Model Building**
   Built a Naive Bayes classification model to predict the target class using probability-based learning.

5. **Model Evaluation**
   Evaluated the model using accuracy score, confusion matrix, classification report, and cross-validation.

## Key Observations

* Naive Bayes is simple, fast, and easy to interpret.
* The model works well as a baseline classification approach.
* Proper preprocessing is important for reliable predictions.
* The model uses probability-based assumptions to classify records.
* Class distribution and feature quality directly affect prediction performance.

## Business Insights

* Classification models can help identify patterns in operational datasets.
* Naive Bayes is useful when quick and interpretable predictions are needed.
* The project demonstrates how probability-based models can support business analytics decisions.
* The approach can be extended to logistics, shipping, operations, and risk classification use cases.

## Conclusion

This project demonstrates how the Naive Bayes algorithm can be used for classification analysis. The model provides a simple and efficient baseline for predicting target classes from structured data.

From a business analytics perspective, this project highlights how machine learning can support classification-based decision-making and help organizations understand patterns in operational data.
